⚡ Prompt Engineering 101

❌ Bad Prompt

"Buatin konten social media"

Terlalu vague, tidak ada konteks

✅ Good Prompt

"Buatkan 5 post Instagram untuk campaign kesehatan mental remaja. Target audience: 16-22 tahun. Tone: supportive tapi tidak menggurui. Include: 1 quote inspiratif, 1 tips praktis, 1 statistik menarik. Format: caption + hashtag."

Spesifik, ada konteks, jelas deliverable

💡 Pro Tips:

✅ Be specific about context
✅ Define target audience
✅ Specify format & length
✅ Give examples if needed
